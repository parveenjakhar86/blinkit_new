# Blinkit-like App (MVC Structure)

## Backend
- Node.js + Express
- MongoDB Atlas
- Folders: model, controller, route

## Frontend
- React (no Vite)

## Structure
- backend/
  - app.js
  - model/
  - controller/
  - route/
- frontend/
  - src/
  - public/

## Setup
1. Add your MongoDB Atlas connection string to backend/.env
2. Run `npm install` in both backend and frontend folders
3. Start backend: `npm run dev` (in backend)
4. Start frontend: (add React scripts or use Create React App)
